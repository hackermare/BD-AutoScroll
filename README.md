# AutoScroll
AutoScroll is a plugin for [BetterDiscord](https://betterdiscord.app/) that brings the mouse wheel button autoscroll feature to the GNU/Linux and macOS client. This plugin is a fork of the [AutoScroll extension made by Pauan](https://github.com/Pauan/AutoScroll), licensed under the [X11/MIT License](https://gitlab.com/programmerpony/BD-AutoScroll/-/raw/main/LICENSE).

![Demo](https://cdn.discordapp.com/attachments/862296245922037800/934057299729870868/autoscroll.gif)

# Installation
If you don't have BetterDiscord installed, follow [this guide](https://github.com/BetterDiscord/BetterDiscord/wiki/Installation).

Once you have BetterDiscord, [download the plugin](https://github.com/programmer-pony/BD-AutoScroll/releases/latest) and move it to the plugins folder.
